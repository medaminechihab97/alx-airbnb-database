# database-script-0x01

This directory contains the initial DDL to create our Airbnb-style database.

## Files

- **schema.sql**  
  Defines all tables, primary keys, foreign keys, check constraints, and indexes:
  1. `User`  
  2. `Location`  
  3. `Property`  
  4. `Booking`  
  5. `Payment`  
  6. `Review`  
  7. `Message`
